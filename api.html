CheckResult Class

Represents the result of a single CIS check.

Attributes

  • check_id - Check identifier (e.g., "1.1.1")
  • title - Check title
  • status - Result status (pass, fail, skip, error)
  • severity - Severity level (low, medium, high, critical)
  • remediation - Fix instructions
